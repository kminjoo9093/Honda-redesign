## 목표
1. 사용자 편의성을 고려한 UI 개선
2. GSAP 활용하여 인터랙티브 웹 사이트 제작
3. 키보드 탭 포커스 관련 접근성 고려
<br><br><br>

## 사용 스킬
- HTML
- CSS
- JAVASCRIPT
- JQUERY
- GSAP
- SWIPER

<br><br><br>
